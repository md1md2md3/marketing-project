# 2. Description du projet e‑commerce (Architecture & Fonctionnalités)

## 2.1 CMS & Principes

- Plateforme: Wix (sans code) — stratégie « Lazy Excellence ».

## 2.2 Architecture des pages

- Accueil: héro vidéo + manifeste + diagramme **AURA Unified** + CTA Services.

- Tier 2: acheter **AURA UNIFIED Kit** (bouton central du diagramme).
- Tier 3: choisir **Bundles par Pilier** (nœuds orbitaux).

- Wix Members: programme **Green Points** (x2 sur SOURCE), statuts Bronze/Argent/Or.
- Wix Automations: paniers abandonnés (J+1/J+3), **Welcome Home** post achat/réservation.
- Privacy & Sécurité: **Data Sovereignty** (stockage local, firewall, segmentation réseau, opt‑in, droit à l’effacement).
